How to back up your wallet?

Use the following instructions to create a backup of your wallet.

  1. 1) Close your wallet.
  2. 2) Openthefolder “%APPDATA%\pivotalcoin\”.
  3. 3) Copy the file “wallet.dat” to a safe location.
  4. 4) Please remember to store this "wallet.dat" in your memory stick, cloud server etc, make sure that you encrypt this, so that noone can easily see the information in this file as it is vital for your assets and remember to decrypt and then export your private key from the cloud server, memory stick etc to the new Pivotalcoin wallet. Your coins should be restored after synchronization with the node.
